# novo-projeto

> Cria pasta isolada pra um cliente novo.

## Fluxo
1. Pergunte o nome do cliente e o segmento.
2. Crie dados/<cliente>/ com um briefing.md em branco estruturado.
3. Registre o cliente como "em foco" em _memoria/estado-atual.md.
4. Pergunte se já roda /montar-gmn ou só coleta briefing.

## Briefing padrão
Nome, segmento, cidade/bairro, cobertura, palavras-chave, carro-chefe,
produtos, serviços, diferenciais, tempo de mercado, dados do dono.
