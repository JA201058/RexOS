# Salvar no GitHub

Versione o estado atual do projeto.

Execute:
1. `git add -A`
2. `git commit -m "..."` com uma mensagem curta e descritiva do que mudou na sessão
3. `git push`

Se o push for rejeitado (fetch first), rode `git pull origin main --no-rebase`
e depois `git push` de novo. Confirme ao final que subiu.
