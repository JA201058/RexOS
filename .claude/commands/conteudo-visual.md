# conteudo-visual

> Cria peças visuais (carrossel, post, story) com a identidade do negócio.

## Identidade
Leia identidade/identidade-visual.md. Use SEMPRE a paleta, fontes e padrões
do negócio que instalou o sistema — nunca um padrão genérico.

## Fluxo
1. Pergunte o tipo (carrossel, post único, story) e o tema.
2. Pergunte o formato se não for óbvio (1080×1350 retrato, 1080×1080 quadrado).
3. Estruture: gancho → desenvolvimento → CTA.
4. Gere via Python/Pillow ou HTML/CSS no formato pedido.
5. Salve em saidas/ e entregue.

## Tom
O tom de voz vem de identidade/. Respeite o público do negócio (B2B, B2C, etc.).
