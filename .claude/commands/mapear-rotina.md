# Mapear rotina

Transforme uma tarefa repetitiva num novo comando.

1. Pergunte qual processo a pessoa repete manualmente.
2. Levante o passo a passo junto com ela.
3. Documente como um novo arquivo de comando em `.claude/commands/`.
4. Confirme o nome do comando e teste a lógica.
5. Sugira /salvar para versionar.
