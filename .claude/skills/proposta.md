# Skill: /proposta

> Proposta comercial em HTML, deployável via Netlify Drop.

## Fluxo
1. Colete: cliente, segmento, escopo (setup / gestão mensal), diferenciais.
2. Monte HTML único com a identidade Rex (navy + dourado, tipografia da marca).
3. Estrutura: capa → diagnóstico → solução → entregáveis → fases (setup +
   recorrente) → prova social (+300 negócios) → CTA.
4. Numere a proposta (ex: #REX-2026).
5. Salve em saidas/ e instrua o deploy no Netlify Drop.

## Regra
Sem preço fixo no template — deixe campo pra preencher por engajamento.
